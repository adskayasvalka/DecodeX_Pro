## 🔍 Что это?

DecodeX Pro объединяет шифрование, хэширование, азбуку Морзе и сканер веб-уязвимостей в одном приложении. Больше не нужно открывать кучу сайтов — всё работает локально и быстро.

---

## 🚀 Возможности

### 🔐 Шифрование
| Алгоритм | Описание |
|----------|----------|
| **AES-256** | Военный стандарт шифрования с ключом |
| **Fernet** | Симметричное шифрование на основе AES |
| **XOR** | Простое XOR-шифрование с паролем |
| **Base64** | Кодирование в читаемый формат |
| **ROT13** | Классический сдвиговый шифр |

### #️⃣ Хэширование
- **MD5** — быстрый, 128 бит
- **SHA-1** — 160 бит
- **SHA-256** — надёжный, 256 бит
- **SHA-512** — максимальная защита, 512 бит
- **Bcrypt** — с солью, устойчив к перебору

### ••• Азбука Морзе
- 🇷🇺 Полная русская азбука (А-Я)
- 🇬🇧 Английская азбука
- 🔢 Цифры и спецсимволы
- ↔️ Конвертация в обе стороны

### 🌐 Сканер сайтов
- 📡 **IP-адрес** сервера
- 🔌 **Открытые порты** (21, 22, 80, 443, 3306 и др.)
- 🔒 **SSL-сертификат** — проверка срока действия
- 📋 **HTTP-заголовки** — полный вывод
- 💉 **SQL-инъекции** — 3 вида payload'ов
- 🛡️ **XSS** — проверка на межсайтовый скриптинг
- 📝 **Пояснения** к каждой найденной уязвимости
