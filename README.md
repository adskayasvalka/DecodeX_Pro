# DecodeX_Pro
DecodeX Pro — мощный крипто-инструмент и сканер безопасности. Шифрование (AES-256, Fernet, XOR, Base64, ROT13), хэши (MD5, SHA-1, SHA-256, SHA-512, Bcrypt), русская и английская азбука Морзе с полной поддержкой символов, веб-сканер (IP-адрес, открытые порты, SSL-сертификат, HTTP-заголовки, проверка SQLi и XSS уязвимостей с пояснениями).
